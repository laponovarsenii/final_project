Проект для поиска фильмов в базе данных MySQL (Sakila) 
с двумя интерфейсами: веб-приложение на Flask и консольное меню. 
Поддерживается поиск по ключевому слову, по жанру и диапазону годов, постраничный вывод результатов. 
Выполненные поисковые запросы логируются в MongoDB, а раздел статистики показывает топ популярных и последние уникальные запросы. 
В составе проекта — модули для работы с MySQL и MongoDB, шаблоны HTML и статические файлы.
